This is an open-source tool that does not use any server or storage to store your personal files. All processing is done between you and your browser, allowing you to load the page and turn off your internet connection freely.

Please note that, because it is browser-based, it uses processing directly in your browser. This may result in lag during processing, so we suggest using a high-end PC for bulk processing. Mobile devices may also experience lag; if this happens, please wait for the page to finish processing.
